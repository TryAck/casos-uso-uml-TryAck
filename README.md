# Diagramas de Casos de Uso

Este repositorio contiene dos ejercicios de diagramas de casos de uso. Por el enunciado del ejercicio, en la raiz del repositorio existira un PDF que contendra ambos diagramas, pero tambien estaran por separado en su carpeta correspondiente.

<details>
    <summary>Los diagramas fueron diseñados usando <a href="https://plantuml.com/es/">PlantUML</a>, un generador de diagramas UML que usa texto plano.</summary>
    <p>PlantUML fue usado porque <a href="https://mermaid.ai/open-source/">Mermaid</a> (El generador que use en el anterior ejercicio, el cual no fue subido a Github) no soporta diagramas de casos de usos.</p>
</details>
